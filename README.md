<details>
<summary><b>⚙️ One-time setup — required for the animation above to render</b></summary>
<br>

The snake graphic is generated by a GitHub Action that reads your contribution graph and publishes it to an `output` branch. Set it up once:

1. In this repository, create a file at `.github/workflows/snake.yml` with the contents below.
2. Go to **Settings → Actions → General → Workflow permissions** and enable **"Read and write permissions."**
3. Go to the **Actions** tab → select **"Generate Snake Animation"** → click **"Run workflow"** to trigger the first run.
4. This creates the `output` branch that the image above pulls from, and re-runs automatically once a day and on every push to `main`.

```yaml
name: Generate Snake Animation

on:
  schedule:
    - cron: "0 0 * * *"   # runs once a day
  workflow_dispatch:       # allows manual runs from the Actions tab
  push:
    branches:
      - main               # regenerate on every push to main

jobs:
  generate:
    permissions:
      contents: write
    runs-on: ubuntu-latest
    steps:
      - name: Generate snake animation
        uses: Platane/snk@v3
        id: snake-gif
        with:
          github_user_name: Luuuucifeeeer
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - name: Push snake animation to output branch
        uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

</details>
